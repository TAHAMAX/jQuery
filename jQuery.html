Qu'est-ce que jquery ?

jQuery est une bibliothèque de fonctions en JavaScript.

C'est une librairie léqère : " écrire mois - faire plus "

Cette bibilothéque contient les fonctionnalités suivantes :

Sélection des éléments HTML

* Manipulation du CSS

* Fonction d'événements HTML

* Effets JavaScript et animations

* Parcours et modification du DOM

* Ajax

jQuery : syntaxe

Avec Jquery, vous sélectionnez des éléments HTML et vous effectuez des actions

Exemples :

$(this).hide(); pour cacher l'élément HTML courant.

$(this).show(); pour afécher l'élément HTML courant.

$("p").hide(); pour cacher tous les éléments HTML de type p.

$("#test").hide(); pour cacher l'élément HTML dont l'ID est #test.

$(".test").hide(); pour cacher tous les éléments possédant la Class "tesd"
